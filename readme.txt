== Description ==

Create beautiful puzzle effect on your images. [Watch demos here](https://www.info-d-74.com/en/image-puzzle-pro-plugin-wordpress-demos-2/)
Pro version with more options here : [https://www.info-d-74.com/en/produit/image-puzzle-pro-plugin-wordpress-2/](https://www.info-d-74.com/en/produit/image-puzzle-pro-plugin-wordpress-2/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Image Puzzle menu to create generate shortcode
4. Paste shortcode in any page or post!
